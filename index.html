Información General

La misión de Software Carpentry es ayudar a los científicos e ingenieros a ser más productivos enseñándoles destrezas básicas de computación como el diseño de programas, control de versiones, gestión de datos y automatización de tareas. Este taller práctico de dos días cubre los conceptos y herramientas básicas; se animará a los participantes a ayudarse unos a otros y para aplicar lo que han aprendido a sus propios problemas de investigación.

Quién: La actividad está dirigido a estudiantes universitarios con interés de mejorar sus destrezas en herramientas computacionales para la ciencia reproducible.

Programa de Estudios

Programando en Python

  • Uso de bibliotecas
  • Trabajando con arreglos
  • Leyendo y graficando datos
  • Creando y usando funciones
  • Bucles y condicionales: for, if, else, ...
  • Programación defensiva
  • Usando Python desde la línea de comandos
  • Referencia...

Programando en R

  • Trabajo con vectores y data frames
  • Leyendo y graficando datos
  • Creando y usando funciones
  • Bucles y condicionales: for, if, else
  • Usando R desde la línea de comandos
  • Referencia...

Control de versiones con Git

  • Creando un repositorio
  • Seguimiento de cambios en los archivos: add, commit, ...
  • Visualizar cambios: status, diff, ...
  • Ignorando archivos
  • Trabajando en la web: clone, pull, push, ...
  • Resolviendo conflictos
  • Licencias Abiertas
  • Donde hospedar el trabajo, y porque
  • Referencia...

Manejando datos con SQL

  • Leyendo y ordenando datos
  • Filtros con where
  • Cálculo de nuevos valores sobre la marcha
  • Manejo de valores faltantes

  • Combinar información de múltiples tablas utilizando join
  • Creación, modificación y eliminación de datos
  • Programación con bases de datos
  • Referencia...
